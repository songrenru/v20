社区管理APP
===============


## 主要功能

* 用户各个角色登录            LoginController
* 首页各个数据显示和对应入口  IndexController
* 收银台                     CashierController
* 工作台                     WorkBenchController
* 个人中心                   MyController
* 业主家属租客相关            UserController
* 业主家属租客审核相关        UserCheckController
* 蓝牙门禁相关                BluetoothDoorController
